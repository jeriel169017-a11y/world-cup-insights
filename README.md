# ⚽ World Cup Insights

## 📖 Descripción

**World Cup Insights** es una aplicación desarrollada en Python para analizar los partidos históricos de la Copa Mundial de la FIFA.

El proyecto aplica Programación Orientada a Objetos (POO), análisis exploratorio de datos (EDA), visualización de datos y un dashboard interactivo desarrollado con Streamlit para facilitar la exploración de la información.

---

## 🎯 Objetivos

- Analizar los partidos históricos de la Copa Mundial de la FIFA.
- Aplicar Programación Orientada a Objetos (POO).
- Realizar procesos de limpieza y análisis exploratorio de datos (EDA).
- Crear visualizaciones mediante gráficos.
- Desarrollar un dashboard interactivo con Streamlit.
- Utilizar Git y GitHub para el control de versiones.

## ▶️ Instalación

Clonar el repositorio:

```bash
git clone https://github.com/jeriel169017-a11y/world-cup-insights.git
```

Ingresar al proyecto:

```bash
cd world-cup-insights
```

Instalar dependencias:

```bash
pip install -r requirements.txt
```

---

## ▶️ Ejecución

Ejecutar la aplicación de consola:

```bash
python main.py
```

Ejecutar el dashboard:

```bash
streamlit run app/app.py
```
