# ⚽ World Cup Insights

## 📖 Descripción

World Cup Insights es una aplicación desarrollada en Python para analizar los partidos históricos de la Copa Mundial de la FIFA.

El proyecto permite descargar automáticamente los datos, realizar procesos de limpieza y análisis exploratorio (EDA), consultar estadísticas relevantes y visualizar la información mediante gráficos y un dashboard interactivo desarrollado con Streamlit.

---

## 🎯 Objetivos

- Analizar los partidos históricos de la Copa Mundial.
- Aplicar Programación Orientada a Objetos (POO).
- Implementar análisis exploratorio de datos (EDA).
- Crear visualizaciones con Matplotlib.
- Desarrollar un dashboard interactivo con Streamlit.
- Gestionar el proyecto mediante Git y GitHub.

## 🚀 Funcionalidades

- Descarga automática del dataset.
- Filtrado de partidos de la Copa Mundial.
- Consulta por selección.
- Consulta por año.
- Consulta por país sede.
- Estadísticas generales.
- Ranking de selecciones con más victorias.
- Promedio de goles por partido.
- Análisis exploratorio de datos (EDA).
- Visualización mediante gráficos.
- Dashboard interactivo con Streamlit.

---

## 📊 Dashboard

El dashboard permite:

- Consultar estadísticas generales.
- Seleccionar una selección nacional.
- Filtrar por edición del Mundial.
- Filtrar por país sede.
- Visualizar tablas y gráficos interactivos.

## ▶️ Ejecución

Clonar el repositorio:

```bash
git clone https://github.com/jeriel169017-a11y/world-cup-insights.git
```

Instalar dependencias:

```bash
pip install -r requirements.txt
```

Ejecutar el proyecto principal:

```bash
python main.py
```

Ejecutar el dashboard:

```bash
streamlit run app/app.py
```
